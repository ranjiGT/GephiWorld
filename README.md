# `GephiWorld`
Some Network Analysis made using Gephi. 
- Plugins used: `GeoLayout`, `NoverlapLayout`, `Multimode Networks Transformation`
- Algorithms used: `Fruchterman Reingold`, `Force Atlas 2`
